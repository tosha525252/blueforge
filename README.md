# BlueForge Labs Website

A responsive React + Vite website concept for BlueForge Labs, inspired by the clear visitor-routing and spacious editorial structure of Lucid Day, but redesigned with BlueForge's dark, intelligent, circuit-forward identity.

## Included

- Homepage with three-path visitor gateway
- BlueForge 8AM product page
- BlueForge Architect product page
- Early-access/contact page
- Responsive navigation
- Reusable logo and UI components
- Custom design system in `src/styles.css`
- No stock photos or paid assets

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

## Notes

- The contact form is visual only. Connect it to Formspree, HubSpot, Supabase, or another backend.
- Replace the placeholder copy and links as your launch details are finalized.
- The logo is an original inline SVG and can be replaced with your final brand asset later.
- Google Fonts are loaded from the web. To self-host fonts, update the first line of `src/styles.css`.
